## Key features

TableCalendar provides everything you need to build sophisticated calendar interfaces:

- **Extensive API** - Easy to use yet powerful enough for complex use cases
- **Preconfigured UI** - Beautiful default styling with extensive customization options
- **Custom builders** - Override any UI element with your own widgets for unlimited design possibilities
- **Locale support** - Built-in internationalization for any language
- **Selection modes** - Single day, range selection, and multiple selection support
- **Dynamic events** - Attach events and holidays to any date
- **Vertical autosizing** - Fit the content or fill the viewport automatically
- **Multiple formats** - Switch between month, two weeks, and week views
- **Swipe boundaries** - Control navigation with customizable first and last days
